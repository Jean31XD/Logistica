🔍 Test de Endpoints - Gestión de Imágenes

Test 1: Endpoint Mínimo (Sin Dependencias)

Este endpoint NO requiere autenticación ni dependencias. Si falla, el problema es de routing o configuración del servidor.

Test 2: Endpoint de Subida (CON Dependencias)

Este endpoint requiere autenticación. Veremos qué logs genera.

Test 3: Verificar Logs de PHP

Después de ejecutar los tests anteriores, revisa los logs:

  • Windows (XAMPP): C:\xampp\php\logs\php_error_log
  • Linux: /var/log/apache2/error.log o /var/log/php/error.log

Busca líneas que contengan "subir_imagen.php" o "test_endpoint.php"
